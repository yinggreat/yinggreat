## About Me

我曾在北京的互联网大厂里写了很多年代码，也在高压、高密度协作和快速迭代的环境里，经历过一个程序员职业生涯最典型的成长路径。

大厂让我看见了技术的力量，也让我更早看见另一面：  
当组织越来越庞大、节奏越来越快，技术有时会被裹挟进绩效、博弈和内耗，慢慢偏离了它原本最迷人的地方。

这些年走下来，我越来越确定：

**技术不该只是少数人向上竞争的筹码。**  
**它也应该成为普通人改变命运、提升效率、获得自由的工具。**

所以，我选择从熟悉的大厂轨道里走出来，遵从自己的内心，专注做一件真正值得长期投入的事：

**做一个 Python 传道者。**

## My Teaching Philosophy

我始终相信，真正好的技术教学，应该做到这几件事：

- 讲人话，而不是堆术语
- 讲路径，而不是甩资料
- 讲项目，而不是只讲语法
- 讲本质，而不是制造焦虑

技术传播不该只是展示“我有多强”，  
更重要的是帮助更多人相信：**自己也可以学会，也可以做出来。**

## Why I Keep Sharing

离开大厂之后，我更想把时间放在长期有价值的事情上。

写代码当然重要，  
但比写出一段代码更有意义的，是帮助更多人跨过门槛，建立信心，真正拥有一门可以改变生活的技能。

如果说过去的 15 年，我主要在系统里解决问题；  
那么接下来的时间，我更愿意把经验沉淀成方法，把方法打磨成内容，把内容交到真正需要它的人手里。

这，就是我继续分享 Python 的原因。
